# Golympus
## Dependency of BLW - Bitcoin and Lightning Wallet

### Dependencies
- Go v1.22.0+
- Make

### Build
```bash
git clone https://github.com/mrbitcoiner/golympus
cd golympus
make
```

### Config
* Copy and change .env parameters accordingly with your environment
```bash
cp .env.example .env
```

### Run
```bash
./out/golympus
```
